# Todo Application - Client

This is the frontend application for the Todo app, built with React and Vite.

## 🚀 Tech Stack

- **Framework:** React
- **Language:** TypeScript
- **Build Tool:** Vite
- **UI Library:** HeroUI
- **Styling:** Tailwind CSS
- **Data Fetching:** Axios
- **Routing:** React Router

## 💻 Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (v16 or higher)
- Yarn package manager

### Installation & Running

```bash
# Install dependencies
yarn install

# Start the development server
yarn run dev
```

The client will be available at `http://localhost:5173`.

## 📝 Environment Variables

> **Note:** For demo purposes, the `.env` file has been included in this directory. In production, this should never be committed to version control.

The client is configured to connect to the backend API. Check the `.env` file for API endpoint configuration.

## 🔗 Backend Connection

This frontend application connects to the NestJS backend API. Make sure the server is running at `http://localhost:3000` before starting the client.

Refer to the main README or the server README for backend setup instructions.
